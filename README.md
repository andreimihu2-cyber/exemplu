# exemplu

1. O resursă este un loc sau serviciu din campus, cum ar fi biblioteca sau cantina.
2. Exemplu URI: /index.html – pagina principală. /biblioteca.html – pagina bibliotecii. #sus este fragment link care sare sus în pagină.
3. Părți statice: paginile HTML și textul de pe site. Părți dinamice: lista de resurse încărcată cu JavaScript din JSON.
